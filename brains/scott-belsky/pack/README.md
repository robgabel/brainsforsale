## What This Is

A structured knowledge graph of Scott Belsky's thinking, extracted from 77 editions of the Implications newsletter. Load as context into any LLM — Claude, ChatGPT, Cursor, or any AI tool that accepts system prompts.

## Quick Start

1. Load `brain-context.md` into your AI tool's context window
2. Run any of the 10 thinking skills: `/advise`, `/teach`, `/debate`, `/connect`, `/evolve`, `/apply`, `/mashup`, `/brainfight`, `/deep-dive`, `/surprise`

```
/advise "I'm thinking about whether to focus my company on serving large enterprises or build a platform for creators. What would Scott think?"
```

## How to Use

**For Claude Code / Cowork:** Skills load automatically. Just invoke `/advise`, `/teach`, etc.

**For Cursor:** Add `brain-context.md` to `.cursor/rules` or project instructions.

**For ChatGPT / Claude.ai:** Paste `brain-context.md` as your first message or system prompt.

## Skill Workflows

Chain skills for deeper results: Decide (`/advise` → `/debate` → `/apply`), Learn (`/teach` → `/deep-dive` → `/evolve`), Create (`/surprise` → `/connect` → `/mashup`), Research (`/deep-dive` → `/mashup` → `/brainfight`).

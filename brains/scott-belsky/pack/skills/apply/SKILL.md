# /apply — Actionable Frameworks

Requires: brain-context.md loaded in session.

Identify the framework (named or inferred). Find atoms that define it. Walk through applying it step-by-step to the user's specific context. Produce concrete output: checklists, matrices, roadmaps, playbooks.

**Output:** Framework name + core principle (original_quote) → Step-by-step application to user's situation → Action plan (checklist/matrix/roadmap) → Key tradeoffs → Success criteria → Suggested follow-up (/debate to stress-test the plan, /deep-dive for full framework context).

**Rules:** Produce something actionable — not theory. Voice first for framework definition. Show tradeoffs. If framework has <3 supporting atoms, note limited grounding.
